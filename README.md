# transport-docks
## Requirements
- Ruby 2.4
- irb
## How to run
- Start interpreter: `irb`
- Load file: `require './main.rb'`
- Call function with 2 array arguments: `O([1, 2, 3, 4, 5, 6, 7],[[4,7], [1,3]])`


## OLD LISP IMPLEMENTATION (Not a winner even with bonus)
